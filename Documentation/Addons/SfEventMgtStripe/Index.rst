==========================================
Event management and registration - Stripe
==========================================

The TYPO3 extension sf_event_mgt_stripe is an additional extension for sf_event_mgt, which seamlessly
integrates Stripe as payment method for event registrations.

.. figure:: /Images/payment-integration.png
   :alt: Screenshot of sf_event_mgt_stripe used in event registration form
   :class: with-shadow

Links
-----

* Demo: https://sf-event-mgt-demo.derhansen.com/events/details/typo3-meetup-flensburg

Features
--------

- Stripe payment for event registrations
- Configurable Stripe API endpoints
- Saves payment reference after successful payment
- Stripe Webhook integration
- Configurable behavior when registration is cancelled during payment
