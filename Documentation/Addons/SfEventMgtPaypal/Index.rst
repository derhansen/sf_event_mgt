==========================================
Event management and registration - PayPal
==========================================

The TYPO3 extension sf_event_mgt_paypal is an additional extension for sf_event_mgt, which seamlessly
integrates PayPal as payment method for event registrations.

.. figure:: /Images/payment-integration.png
   :alt: Screenshot of sf_event_mgt_paypal used in event registration form
   :class: with-shadow

Links
-----

* Demo: https://sf-event-mgt-demo.derhansen.com/events/details/typo3-meetup-flensburg

Features
--------

- PayPal payment for event registrations
- Support for Sandbox and Production environment
- Uses official PayPal `paypal-server-sdk`
- Saves payment reference after successful payment
- Configurable behavior when registration is cancelled during payment
