========================================
Demo website for sf_event_mgt and addons
========================================

A demo website to demonstrate features for **sf_event_mgt** and addons:
👉 https://sf-event-mgt-demo.derhansen.com/

This site showcases a fully working installation of the extension with realistic sample data. It’s a great way to
*see how events and registrations are presented in the frontend*.

.. figure:: /Images/sf_event_mgt_demo.png
   :alt: Screenshot of demo website
   :class: with-shadow

What You Can Explore
--------------------

Frontend Features
~~~~~~~~~~~~~~~~~
- List of upcoming events
- Event detail page
- Event Registration form
- Event calendar
- Event search page

Commercial Addons
~~~~~~~~~~~~~~~~~
- Multiple person registration (:ref:`sf_event_mgt_multireg <sfeventmgtmultireg>`)
- PayPal payment integration (:ref:`sf_event_mgt_paypal <sfeventmgtpaypal>`)
- Stripe payment integration (:ref:`sf_event_mgt_stripe <sfeventmgtstripe>`)
